    Projection onto the L1-Norm Ball

    12 minute read

    Published: February 10, 2023

    In this post we will discuss a frequently visited problem in convex optimization: projection onto the ℓ1ℓ1-norm ball. I think this is a great example to understand the concepts of duality and piecewise functions and the phenomenon called sparsity, where the solution to a problem contains mostly zeros, and only some remain “activated.”

    The Problem

    Suppose we have a vector a∈Rna∈Rn and want to find another vector xx in an ℓ1ℓ1-norm ball such that the distance between aa and xx is the smallest possible. In other words, we seek to solve the following problem

    minx{f(x)=12∥x−a∥22}such that∥x∥1≤κ.(1)(1)minx{f(x)=12‖x−a‖22}such that‖x‖1≤κ.

    The constraint ∥x∥1≤κ‖x‖1≤κ denotes the ℓ1ℓ1-norm ball of radius κ>0κ>0 (that is, all the points whose ℓ1ℓ1 norm is at most κκ) and f(x)f(x) is the primal objective function.

    Does this problem have a closed-form solution? Yes, but not always. Like all projection problems, if aa is already in the convex set (∥a∥1≤κ‖a‖1≤κ) then the solution is simply x=ax=a, and we are done. Otherwise, we will need to approximate the solution.

    In the above figure, we see an example in n=2n=2 dimensions. The black square (containing all points on and within its boundaries) represents the ℓ1ℓ1-norm ball of radius κ=1κ=1. The center of the blue circle, in red, is the vector aa and the radius of the circle is the smallest distance from aa to the ℓ1ℓ1-norm ball.

    Duality

    It is easy to see that (1)(1) is a convex optimization problem as its objective and constraint are both convex. Note that in this case, x=0x=0 is a strictly feasible solution, which means, by Slater’s condition, strong duality holds. We therefore will aim to solve (1)(1) by maximizing its dual function. Let γ≥0γ≥0 be the dual variable; the Lagragian is

    L(x,γ)=12∥x−a∥22+γ(∥x∥1−κ).(2)(2)L(x,γ)=12‖x−a‖22+γ(‖x‖1−κ).

    Finding the dual objective function requires us to minimize L(x,γ)L(x,γ) with respect to xx. Notice that we can rewrite the Lagrangian as

    L(x,γ)=−κγ+n∑i=1(12(xi−ai)2+γ|xi|),L(x,γ)=−κγ+∑i=1n(12(xi−ai)2+γ|xi|),

    where the subscript ii denotes the iith element of a vector. If we let

    si(xi,γ)=12(xi−ai)2+γ|xi|,si(xi,γ)=12(xi−ai)2+γ|xi|,

    then minimizing L(x,γ)L(x,γ) with respect to xx means to minimize each si(xi,γ)si(xi,γ) with respect to xixi. Fortunately, the problem minxisi(xi,γ)minxisi(xi,γ) has a unique and closed-form solution:

    xi(γ)=⎧⎨⎩ai−γifai>γ0if−γ≤ai≤γai+γifai<γ.(3)(3)xi(γ)={ai−γifai>γ0if−γ≤ai≤γai+γifai<γ.

    This is is called the soft thresholding operator for γγ. Equation (3)(3) shows us how to convert a dual solution γγ to a primal solution xx. Now, if we let s∗i(γ)=minxisi(xi,γ)=si(xi(γ),γ)si∗(γ)=minxisi(xi,γ)=si(xi(γ),γ), the dual objective is

    g(γ)=minxL(x,γ)=−κγ+n∑i=1s∗i(γ).g(γ)=minxL(x,γ)=−κγ+∑i=1nsi∗(γ).

    We know that gg is a concave function by design. Furthermore, since the solution to minxisi(xi,γ)minxisi(xi,γ) is unique for every γ≥0γ≥0, by Danskin’s theorem, each s∗isi∗ is differentiable, which makes gg differentiable as well. We can easily verify that the derivative of gg is

    g′(γ)=−κ+n∑i=1max(|ai|−γ,0).(4)(4)g′(γ)=−κ+∑i=1nmax(|ai|−γ,0).
    Proof.

    It only remains to be shown that ddγs∗i(γ)=max(|ai|−γ,0)ddγsi∗(γ)=max(|ai|−γ,0). To see why, note that s∗i(γ)=si(xi(γ),γ)=12(xi(γ)−a)2+γ(xi(γ)).si∗(γ)=si(xi(γ),γ)=12(xi(γ)−a)2+γ(xi(γ)). It is easy to show that, by (3)(3), (xi(γ)−a)2=min(|ai|,γ)2,(xi(γ)−a)2=min(|ai|,γ)2, and (xi(γ)−a)2=max(|ai|−γ,0)2.(xi(γ)−a)2=max(|ai|−γ,0)2. Now we consider two cases of γγ. First, if γ≤|ai|γ≤|ai|, we have s∗i(γ)=−12γ2+|ai|γsi∗(γ)=−12γ2+|ai|γ, which gives its derivative equal to |ai|−γ|ai|−γ. Second, if if γ>|ai|γ>|ai|, s∗i(γ)=0si∗(γ)=0. Either way, ddγs∗i(γ)=max(|ai|−γ,0)ddγsi∗(γ)=max(|ai|−γ,0).

    So far we have been able to find the dual function g(γ)g(γ) and its derivative g′(γ)g′(γ). Now we will explore a method to maximize g(γ)g(γ) and recover the primal optimal solution.

    Optimizing the Dual Function

    As a reminder, we will aim to solve the problem

    maxγg(γ)such thatγ≥0.(5)(5)maxγg(γ)such thatγ≥0.

    Since gg is concave and differentiable, we can aim to maximize it by using a hill-climbing algorithm such as gradient ascent with backtracking line search. Below is an example dual function and its derivative at various values of γγ.

    In this post we will solve this problem using a different method called bisection. The aim here is to set the derivative to zero and solve for γγ. In other words, we seek the solution to g′(γ)=0g′(γ)=0. The bisection method requires us to have a range [γmin,γmax][γmin,γmax] in which we are sure the optimal solution γ∗γ∗ lies.

    First, since γ∗γ∗ must be feasible, we set γmin=0γmin=0. To find an upper bound, note that since the optimal objective value for Problem (1)(1) must be non-negative, and strong duality holds, the optimal value for Problem (5)(5) is also non-negative. This implies that

    −κγ∗+n∑i=1si(xi(γ∗),γ∗)≥0.−κγ∗+∑i=1nsi(xi(γ∗),γ∗)≥0.

    Therefore,

    γ∗≤1κn∑i=1si(xi(γ∗),γ∗)≤1κn∑i=1si(0,γ∗)=1κn∑i=1a2i2=12κ∥a∥22,γ∗≤1κ∑i=1nsi(xi(γ∗),γ∗)≤1κ∑i=1nsi(0,γ∗)=1κ∑i=1nai22=12κ‖a‖22,

    where the second inequality is due to the fact that xi(γ∗)xi(γ∗) is the minimizer if si(x,γ∗)si(x,γ∗). So an upper bound we can set for γ∗γ∗ is γmax=12κ∥a∥22γmax=12κ‖a‖22.

    Now that we know γ∗γ∗ is in between γmin=0γmin=0 and γmax=12κ∥a∥22γmax=12κ‖a‖22, the bisection method works as follows. First, let γ=(γmin+γmax)/2γ=(γmin+γmax)/2. If the sign of g′(γ)g′(γ) is the same as that of g′(γmin)g′(γmin), then γminγmin is updated to γγ. Otherwise, γmaxγmax is updated to γγ. It is simple as that! The method is also guaranteed to converge, as after each iteration, the length of the interval [γmin,γmax][γmin,γmax] is reduced by half.

    Another point to note is that g′g′ is a monotonically non-increasing function. It achieves a minimum of −κ−κ when γ≥maxi{|xi|}γ≥maxi{|xi|} and a maximum of −κ+∥a∥1−κ+‖a‖1 when γ≤mini{|xi|}γ≤mini{|xi|}. When ∥a∥≤κ‖a‖≤κ, g′g′ is always negative so the solution can be incorrect. In this special case, we can directly conclude the solution x=ax=a without having to solve anything else.

In the left plot, κκ is set to 2<∥a∥1=62<‖a‖1=6, which allows g′g′ to cross the y=0y=0 line and so the solution to g′(γ)g′(γ) exists. On the other hand, in the right plot where κκ exceeds ∥a∥1‖a‖1, no solution exists. In this case one can output aa as the solution already.

    Implementation

    Here is a simple Python implementation of the bisection method for maximizing the dual objective. First we define a few functions.

    def primal_fn(x, a):
    return 0.5 * np.sum((x - a) ** 2)

# Vectorize the computation of s_i(x, gamma)
def s(x, gamma, a):
    return 0.5 * (x - a) ** 2 + gamma * np.abs(x)

def x_gamma(gamma, a):
    sol = np.zeros_like(a)
    idx = a > gamma
    sol[idx] = a[idx] - gamma
    idx = a < - gamma
    sol[idx] = a[idx] + gamma
    return sol

def dual_fn(gamma, kappa, a):
    x = x_gamma(gamma, a)
    return - kappa * gamma + np.sum(s(x, gamma, a))

def dual_grad(gamma, kappa, a):
    return - kappa + np.sum(np.maximum(np.abs(a) - gamma, 0))

    Then, the bisection method is straightforward. We can let the iterations run until the difference γmax−γminγmax−γmin reaches below a pre-defined error εε, at which point the derivative should be close enough to 00.

    def bisection(a, kappa, eps=1e-5):
    gamma_min, gamma_max = 0, (1 / (2 * kappa)) * np.sum(a ** 2)
    # Run until gamma_max and gamma_min are the same
    while gamma_max - gamma_min > eps:
        gamma = (gamma_max + gamma_min) / 2
        grad = dual_grad(gamma, kappa, a)
        if grad < 0: 
            gamma_max = gamma
        else:
            gamma_min = gamma
    return gamma

    The first two plots in this post are produced using the following code.

    # Point to be projected
a = np.array([1.1, 1.2])
# Radius of the ell_1 norm ball
kappa = 1
# Find approximate solution to the dual problem
dual_solution = bisection(a, kappa, eps=1e-5)
# Convert to the primal solution
primal_solution = x_gamma(dual_solution, a)

    Sparseness of the Solution

    In the figure at the top of this post, you probably have observed that as aa moves, there seems to be a “region” of aa in which the solution stays in a vertex of the square. Projection onto the ℓ1ℓ1-norm ball has an interesting characteristic: in high dimensions, the optimal solution has a tendency to be sparse, which means most of its elements are driven to zero. To see how, let’s try an example.

    >>> np.random.seed(100)
>>> a = np.random.randn(100)
>>> dual_solution = bisection(a, kappa=1, eps=1e-10)
>>> primal_solution = x_gamma(dual_solution, a)
>>> print(np.count_nonzero(primal_solution))
5

    In this example, I generated a 100100-dimensional vector aa by independently sampling 100100 samples from the standard normal distribution. After projection, the solution only contains 55 non-zero values. Only 55 out of 100100 elements remain non-zero! (Be careful: I set εε to be very small but it’s probably better to check equality using np.allclose to compare two floating point numbers.)

    You may ask, “What’s the significance of this?” The tendency to drive most variables to zero is behind the success of the lasso method. Imagine you are performing a regression analysis with many, many variables. The lasso, or ℓ1ℓ1 regularized, problem is

    minw12∥Xw−y∥22+λ∥w∥1,(6)(6)minw12‖Xw−y‖22+λ‖w‖1,

    where XX is the design matrix, yy is the ground-truth labels and λλ is the regularization strength. Note that the formulation of lasso regression is not exactly the problem discussed in this method: the variable has to go through a linear transformation in lasso. However, observations remain similar: the solution w∗w∗ to this problem tends to be sparse where most weights are driven to zero.

    While ℓ2ℓ2 is a more popular regularizer, ℓ1ℓ1 may be preferred if you like to assess features’ importance: those with non-zero coefficients tend to be very few and represent the most important features you may want to keep during feature selection.

    Conclusion

    In this post we explore the problem of projection onto an ℓ1ℓ1-norm ball. We formalize the primal problem and see how the dual problem can be expressed and optimized. We also observe that the solution tends to be sparse in high dimensions.

    Several things deserve some mentioning in these concluding remarks. First, we have yet to talk about the asymptotic complexity of solving ℓ1ℓ1 projection. That is, given some tolerance ϵϵ, how much time do we need to achieve an approximate solution xx to Problem (1)(1) such that f(x)−f(x∗)<ϵf(x)−f(x∗)<ϵ? Second, you may be interested in a variant of ℓ1ℓ1 projection called simplex projection, where the variable xx is also constrained to be non-negative. In this case the Lagrangian in (2)(2) must involve another set of variables for the constraints xi≥0,i=1,…,nxi≥0,i=1,…,n, and a different optimization algorithm is needed. Third, minimizing the lasso objective as in (6)(6) deserves some discussion, too. The resources below should offer some answer to these questions.
